# Proteomics

This folder contains proteomics analysis code added separately from the transcriptomic workflows.

This section was added to avoid changing the existing transcriptomic analysis structure.

## Structure

- `GO_analysis/`: GO enrichment bubble plot scripts, input tables, and outputs
- `IP_analysis/`: IP rerun analysis script, input Excel file, and outputs
- `heatmap/`: heatmap scripts and input tables for TRP intensity plots
- `proteomics_figures/`: additional proteomics figure scripts and supporting input files

## Notes

Where possible, scripts and their required input files are kept together in separate analysis folders so the transcriptomic parts of the repository remain untouched.
