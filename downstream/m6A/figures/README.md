# m6A figures (Nanopore direct RNA)

This folder contains plotting notebooks used to generate figures
for m6A analysis from Nanopore direct RNA sequencing.

All figures are derived from outputs generated by:

- 00_import_modkit_pileup_to_parquet.py
- 01_attach_motifs.py
- 02_call_modified_sites_and_export_bed.py
- 03_assign_sites_to_genes.py

Each notebook corresponds to a specific figure or panel in the manuscript.
