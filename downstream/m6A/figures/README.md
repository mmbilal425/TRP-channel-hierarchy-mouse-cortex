## m6A Figures (Nanopore direct RNA; modkit)

This folder contains scripts used to generate publication figures
for the m6A epitranscriptomic analysis.

### Supplementary Figure S5

| Panel | Script | Description |
|------|--------|-------------|
| S5a | `SuppFig_S5a_m6A_stoichiometry_TRP.py` | Distribution of m6A stoichiometry across TRP-associated mRNA sites |
| S5b | `SuppFig_S5b_m6A_ranked_motifs_TRP.py` | Ranked motif analysis of modified m6A sites highlighting DRACH enrichment |
| S5c | *(to be added)* | Number of modified sites per TRP gene |

### Notes
- Scripts expect annotated m6A tables produced by `downstream/m6A/01–03`
- Output figures are written as PDF (300 dpi) for publication
